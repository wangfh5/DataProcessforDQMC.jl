DataProcessforDQMC.format_pair_onsite_r — Method
format_pair_onsite_r(data_dir::String, sigs::Array{Float64,1}, L::Int)

Format the pair_onsite_r.bin in data_dir into a dataframe and store it in a .csv and .toml file. For now, only support one sublattice system. This function support old-fashioned output data, i.e., only output the values without the coordinates, so users need to specify the lattice size L.

format_pair_onsite_r(data_dir::String, sigs::Array{Float64,1})

Format the pair_onsite_r.bin in data_dir into a dataframe and store it in a .csv and .toml file. For now, only support one sublattice system. This function support new-fashioned output data, i.e., output both the coordinates and the values in each line.

DataProcessforDQMC.insert_parameters_cols — Method
insert_parameters_cols(df::DataFrame,data_dir::String)

Exact parameters from data_dir and insert them as columns into df. e.g. data_dir = ".../proj_majo_cy.b16.000.U-4.00.Delta_ppm0.4.mu-0.5.L8.dtau0.05/" will extract beta, U, Delta_ppm, mu, L and dtau from it.
